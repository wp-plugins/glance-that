Adds content control to At a Glance on the Dashboard

== Description ==

Glance That allows users to customize the content viewable in At a Glance on the WordPress Dashboard. Users can add/remove items from At a Glance and assign custom dashicons for their display. Currently, the following items are supported ...

* Custom post types
* Revisions (admins only)
* Media
* Users

Additionally, Glance That allows you to view the number of posts matching the following post statuses:

* Scheduled
* Pending
* Draft
* Private
* Trash
* Unattached (Media)
* Spam (Comments)

Statuses can be hid by adding the following to wp-config.php
`define( GT_SHOW_ALL, FALSE );`

== Installation ==

1. Upload the `glance-that` directory to `/wp-content/plugins/`
1. Activate the plugin through the 'Plugins' menu in WordPress
